Follow these steps to prepare the data required for the Python script.

1. Save the Beatport Playlist

Navigate to the public Beatport playlist you wish to use.

"Save" the entire webpage (CTRL+S / CMD+S) to your Downloads folder. Ensure you save as "Webpage, Complete" so the assets folder is included.

2. Move Files to Project Directory

Locate the saved .html file and its corresponding _files folder in your Downloads.

Copy both the file and the folder into your project's root directory.

3. Generate JSON Data

Upload only the .html file to Gemini.

Copy and paste the prompt below into the chat to extract the data:


|->COPY PASTE AFTER THIS->

I am trying to extract the following information from the given html:
1. Song name with information on whether it is an extended mix or whether it is another artists remix in the same html file, appended into the name of the song directly without any brackets in “song_name”
2. List of artist names in a single field, separated by commas in “artist_name”
3. Label/ Remixer Name in “label_name”
4. Genre in “genre”
5. BPM & Key in “bpm_key”
6. Album Art in “album_art”

This HTML file contains a public Beatport playlist. This is a sample of how the json should be formatted:

[
    {
        "song_name": "Take Your Places Extended Mix",
        "artist_name": "Westend, SIDEPIECE",
        "label_name": "WestendSS",
        "genre": "tech_house",
        "bpm_key": "130 bpm, Fm",
        "album_art": …
    },
….


->UNTIL HERE|

4. Create the JSON File

Copy the JSON output generated by Gemini.

Create a new file in your project directory (e.g., playlist_data.json).

Paste the JSON content into this file and save it.

5. Run the Script

Refer to the main script's README.md for the command to run the script using your new .json file.
